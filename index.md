# QA Hub Public Docs

QA Hub is a self-hosted QA testing platform. The Chrome extension connects to the backend that you deploy and control.

Public documentation:

- [Privacy Policy](./privacy-policy/)
- [Self-Hosting Guide](./self-hosting/)
- [Support](./support/)

If you are installing QA Hub for the first time, start with the Self-Hosting Guide and then connect the Chrome extension to your backend URL.
