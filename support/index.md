# QA Hub Support

For support, contact: `etitkov@bruin.com`

When contacting support, include:

- QA Hub extension version
- QA Hub backend version
- Chrome version
- Whether the issue affects a local or public backend deployment
- Steps to reproduce the problem
- Screenshots, console errors, or API error details if available

QA Hub is self-hosted, so environment details may be required to diagnose backend configuration, authentication, network, or integration issues.
